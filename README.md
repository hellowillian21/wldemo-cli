# wldemo-cli
## Introduction
`wldemo-cli` is a command-line tool for quickly generating HTML+CSS+JavaScript demo projects
## Installation  
```
  npm install -g wldemo-cli
```
## How to use

- Create a new project
  ```
  wldemo new
  ```
  and input your project name and press return, done!

- View version
  ```
  wldemo -v
  ```
